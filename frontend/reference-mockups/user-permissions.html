Permission Logic:

  • View Only: User can view the module but cannot create, edit, or delete records.
  • Allow Edit: User can create and edit records (requires View Only to be checked).
  • Allow Delete: User can delete records (requires View Only and Allow Edit).
  • Allow Export: User can export data to Excel/PDF.
